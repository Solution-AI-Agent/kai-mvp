# 회의록 생성 앱 설계 문서

## 개요

회의 녹음 파일(mp3/wav)을 업로드하면 STT로 텍스트 변환 후, LLM이 회의록을 자동 생성하는 웹 앱.
MVP로 최소 기능만 구현하며, 인증/DB 없이 일회성 처리 방식으로 동작한다.

## 기술 스택

| 항목 | 기술 |
|------|------|
| 프론트엔드 | React + Vite + Tailwind CSS |
| 백엔드 | FastAPI (Python) |
| STT | 로컬 Whisper (openai-whisper) |
| LLM | OpenRouter API |
| 테마 | 다크 배경 + 에메랄드 포인트 컬러 |
| 인증 | 없음 |
| DB | 없음 |

## 사용자 흐름

1. 웹 페이지 접속 (단일 페이지)
2. 오디오 파일(mp3/wav) 업로드
3. "회의록 생성" 버튼 클릭
4. 처리 중 프로그레스 표시 (STT 변환 중 → 회의록 생성 중)
5. 결과 화면에 STT 원문(좌)과 회의록(우) 나란히 표시
6. 마크다운 파일 다운로드

## API 설계

### `POST /api/generate`

오디오 파일을 받아 STT 변환 및 회의록 생성 후 결과를 반환한다.

**Request:**
- Content-Type: `multipart/form-data`
- Body: `file` (오디오 파일, mp3/wav)

**Response:**
```json
{
  "transcript": "STT 변환된 전체 텍스트",
  "summary": "회의 요약",
  "discussions": ["주요 논의사항 1", "주요 논의사항 2"],
  "decisions": ["결정사항 1", "결정사항 2"],
  "actionItems": ["액션아이템 1", "액션아이템 2"],
  "markdown": "전체 회의록 마크다운 문자열"
}
```

**처리 순서:**
1. 오디오 파일 수신 → 임시 저장
2. Whisper로 STT 변환
3. 변환된 텍스트를 OpenRouter LLM에 전달 → 회의록 생성
4. 임시 파일 삭제
5. 결과 반환

## 프론트엔드 레이아웃

```
┌─────────────────────────────────────────┐
│              앱 타이틀                    │
├─────────────────────────────────────────┤
│  [파일 업로드 영역]  [회의록 생성 버튼]    │
├───────────────────┬─────────────────────┤
│                   │                     │
│   STT 원문 (좌)   │   회의록 (우)        │
│                   │   - 요약             │
│                   │   - 주요 논의사항     │
│                   │   - 결정사항          │
│                   │   - 액션아이템        │
│                   │                     │
├───────────────────┴─────────────────────┤
│          [마크다운 다운로드 버튼]          │
└─────────────────────────────────────────┘
```

- 테마: 다크 배경 + 에메랄드(#10B981 계열) 포인트 컬러
- 결과 영역은 파일 처리 완료 후에만 표시

## 회의록 항목

LLM에게 아래 4가지 항목을 구조화하여 생성하도록 프롬프트한다:

1. **요약**: 회의 전체 내용을 2-3문장으로 요약
2. **주요 논의사항**: 회의에서 논의된 핵심 주제들
3. **결정사항**: 회의에서 확정된 사항들
4. **액션아이템**: 후속 조치가 필요한 항목들 (담당자, 기한 포함 가능)

## 제약사항

- 인증 없음: 로컬 또는 단독 사용 목적
- DB 없음: 회의록을 서버에 저장하지 않음, 일회성 처리
- 오디오 형식: mp3, wav 지원
- Whisper 모델: 로컬 실행 (GPU 권장, CPU도 가능)
